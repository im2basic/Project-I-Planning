## Hero Design Conventions

All hero sheets follow this template (approved by user):

1. **Header table** — element, class, role, weapon, rarity, banner pool, status
2. **Appearance** — visual identity, color palette
3. **Stat Grades** — HP/ATK/DEF/SPD/CR/CD/ACC (S/A/B/C/D scale)
4. **Passive** — name, mechanics, max stacks
5. **Basic Attack** — name, effects
6. **Skill 1** — name, cooldown, multiplier, effects
7. **Skill 2** — name, cooldown, multiplier, effects
8. **Ultimate** — name, cooldown, multiplier, effects
9. **Character Identity** — strengths / weaknesses
10. **Chibi Animations** — idle / attack / skill 1 / skill 2 / ultimate / death
11. **Team Role** — description
12. **Recommended Partners** — list
13. **Auto-AI Defaults** — skill 1 / skill 2 / ultimate priority rules
14. **Quote**
15. **Status notes** (WIP heroes only)

See [`../gdd/01-combat.md`](../gdd/01-combat.md) for combat mechanics referenced in skill descriptions, and [`../gdd/02-elements-and-classes.md`](../gdd/02-elements-and-classes.md) for element + class definitions.
