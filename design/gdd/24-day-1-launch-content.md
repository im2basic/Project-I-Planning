## 3. Banner Lineup — Day 1

### Standard Banner (permanent, live at launch)
- 9 heroes (see above), permanent rotation
- Rates: Legendary 0.6% / Epic 7.2% / Rare 92.2%
- Hard pity at pull 50, no soft pity
- Standard Selector unlocks at 210 cumulative pulls
- Free Daily Pull active
- Wishlist (3 picks) active

### Launch Limited Banner — Auron's Frostbound Dawn (live at launch)
- **Hero:** Auron (Water Archer)
- **Duration:** 21 days from launch
- **Featured Epics:** TBD — pick 2-3 from the Epic pool to rate-up (data table)
- **Bundles available:**
  - Auron Starter Bundle ($14.99): 800 Crystals + 10 Limited Tickets + 50 Skill Stones + 1 Reroll Stone
  - Auron Power Bundle ($29.99): 1,800 Crystals + 30 Limited Tickets + 1 Apex Catalyst + 10 Reroll Stones + 1 Pristine Element Crystal
- Pity carries over to next Limited Banner

### Beginner Step-Up Banner (live at launch, account-gated)
- Active for accounts ≤ 14 days old
- 3 steps:
  - Step 1: 10 pulls @ 1,200 Gems — guaranteed Epic+ in batch
  - Step 2: 10 pulls @ 1,400 Gems — guaranteed Legendary in batch (from Standard pool)
  - Step 3: 10 pulls @ 1,600 Gems — guaranteed Standard Selector unlock + 1 free Standard 5★ pick
- Isolated pity counter

## 5. Tower of Trials — Season 1

### Configuration
- 100 floors
- Monthly reset (resets first of each calendar month at 00:00 UTC)
- Hero lock between floors except every 10th floor (Rest Floor)
- Tower Coin earned per floor

### Floor modifier schedule (key floors locked, filler floors generic)

| Floor band | Modifier |
|------------|----------|

| 10 (Boss) | Phase Shift — boss has 2 HP phases |

| 16-19 | None |
| 20 (Boss) | Add Summoning — 2 minions every 4 turns |
| 21-27 | None |
| 28 | **Reverse Element** — element advantage becomes disadvantage |
| 29 | None |
| 30 (Boss) | DPS Check — boss enrages at turn 20 |
| 31-34 | None |
| 35 | **One-Hero Force** — only one hero acts per turn (cycles) |
| 36-39 | None |
| 40 (Boss) | Phase Shift — 3 HP phases |
| 41-46 | None |
| 47 | **Ultimate Lockout** — Ultimates disabled |
| 48-49 | None |
| 50 (Boss) | Phase Shift — 3 phases, each phase = different element |
| 51-54 | None |
